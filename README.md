# SongodaApi

## Installation
### Maven
```xml
<repository>
  <id>sllcoding</id>
  <url>https://repo.sllcoding.dev</url>
</repository>
```
```xml
<dependency>
  <groupId>sll.coding</groupId>
  <artifactId>SongodaAPI</artifactId>
  <version>LATEST</version>
</dependency>
```
