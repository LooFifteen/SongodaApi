# SongodaApi

## Installation
### Maven
```xml
<repository>
  <id>jitpack</id>
  <url>https://jitpack.io</url>
</repository>
```
```xml
<dependency>
  <groupId>com.github.SLLCoding</groupId>
  <artifactId>SongodaApi</artifactId>
  <version>{Version}</version>
</dependency>
```

The version is the latest commit hash shortened to 8 characters.
